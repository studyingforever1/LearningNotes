## Linux 查看日志常用命令

### 1. `tail` — 最常用

```bash
tail -f /var/log/sales.log          # 实时追踪日志（最常用）
tail -n 100 /var/log/sales.log      # 查看最后100行
tail -fn 200 /var/log/sales.log     # 实时追踪，显示最后200行
```

### 2. `grep` — 过滤关键字

```bash
grep "SalesException" /var/log/sales.log              # 过滤SalesException行
grep -n "SalesException" /var/log/sales.log           # 显示行号
grep -i "salesexception" /var/log/sales.log           # 忽略大小写
grep -C 5 "SalesException" /var/log/sales.log         # 匹配行前后各5行（上下文）
grep -A 3 "SalesException" /var/log/sales.log         # 匹配行之后3行
grep -B 3 "SalesException" /var/log/sales.log         # 匹配行之前3行
grep "SalesException\|WARN" /var/log/sales.log        # 多关键字
```

### 3. `tail + grep` — 黄金组合

```bash
tail -f /var/log/sales.log | grep "SalesException"
tail -f /var/log/sales.log | grep -E "SalesException|ERROR|WARN"
```

### 4. `less` — 交互式查看

```bash
less /var/log/sales.log
```

进入后的操作：

| 按键              | 作用                  |
| ----------------- | --------------------- |
| `G`               | 跳到文件末尾          |
| `g`               | 跳到文件开头          |
| `/SalesException` | 向下搜索              |
| `?SalesException` | 向上搜索              |
| `n`               | 下一个匹配            |
| `q`               | 退出                  |
| `F`               | 类似 tail -f 实时追踪 |

### 5. `cat` — 小文件查看

```bash
cat /var/log/sales.log               # 直接输出全部内容
cat -n /var/log/sales.log            # 带行号
```

### 6. `head` — 查看开头

```bash
head -n 50 /var/log/sales.log        # 查看前50行
```

### 7. `awk` — 按列/条件提取

```bash
# 打印第1、3列
awk '{print $1, $3}' /var/log/sales.log

# 过滤包含SalesException的行，打印时间和内容
awk '/SalesException/ {print $1, $2, $0}' /var/log/sales.log

# 统计SalesException出现次数
awk '/SalesException/{count++} END{print count}' /var/log/sales.log
```

### 8. `sed` — 按行范围截取

```bash
# 查看第100到200行
sed -n '100,200p' /var/log/sales.log

# 查看包含SalesException的行
sed -n '/SalesException/p' /var/log/sales.log
```

### 9. 查看压缩日志

```bash
zcat sales.log.gz                                  # 直接输出
zgrep "SalesException" sales.log.gz               # 在压缩包内grep
zless sales.log.gz                                 # 交互式查看
```

### 场景速查

| 场景         | 命令                                       |
| ------------ | ------------------------------------------ |
| 实时看日志   | `tail -f sales.log`                        |
| 实时看报错   | `tail -f sales.log \| grep SalesException` |
| 查最后N行    | `tail -n 100 sales.log`                    |
| 大文件翻页   | `less sales.log`                           |
| 搜索关键词   | `grep -n "SalesException" sales.log`       |
| 看报错上下文 | `grep -C 10 "SalesException" sales.log`    |
| 统计错误数   | `grep -c "SalesException" sales.log`       |
| 查指定行范围 | `sed -n '100,200p' sales.log`              |
| 查压缩日志   | `zgrep "SalesException" sales.log.gz`      |



## Java的JDK跟JVM是什么关系

JDK（Java Development Kit）是一个功能齐全的 Java 开发工具包，供开发者使用，用于创建和编译 Java 程序。它包含了 JRE（Java Runtime Environment），以及编译器 javac 和其他工具，如 javadoc（文档生成器）、jdb（调试器）、jconsole（监控工具）、javap（反编译工具）等。

JRE 是运行已编译 Java 程序所需的环境，主要包含以下两个部分：

1. **JVM** : 也就是我们上面提到的 Java 虚拟机。
2. **Java 基础类库（Class Library）**：一组标准的类库，提供常用的功能和 API（如 I/O 操作、网络通信、数据结构等）。

简单来说，JRE 只包含运行 Java 程序所需的环境和类库，而 JDK 不仅包含 JRE，还包括用于开发和调试 Java 程序的工具。



## 基本数据类型long有几个字节

这 8 种基本数据类型的默认值以及所占空间的大小如下：

| 基本类型  | 位数 | 字节 | 默认值  | 取值范围                                                     |
| :-------- | :--- | :--- | :------ | ------------------------------------------------------------ |
| `byte`    | 8    | 1    | 0       | -128 ~ 127                                                   |
| `short`   | 16   | 2    | 0       | -32768（-2^15） ~ 32767（2^15 - 1）                          |
| `int`     | 32   | 4    | 0       | -2147483648 ~ 2147483647                                     |
| `long`    | 64   | 8    | 0L      | -9223372036854775808（-2^63） ~ 9223372036854775807（2^63 -1） |
| `char`    | 16   | 2    | 'u0000' | 0 ~ 65535（2^16 - 1）                                        |
| `float`   | 32   | 4    | 0f      | 1.4E-45 ~ 3.4028235E38                                       |
| `double`  | 64   | 8    | 0d      | 4.9E-324 ~ 1.7976931348623157E308                            |
| `boolean` | 1    |      | false   | true、false                                                  |
