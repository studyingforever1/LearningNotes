### 冒泡排序

反复比较相邻元素，若顺序错误则交换，每轮将最大值"冒泡"至末尾。加入 `swapped` 标志可在数组已有序时提前终止。

```java
public static void bubbleSort(int[] arr) {
    int n = arr.length;
    for (int i = 0; i < n - 1; i++) {
        boolean swapped = false;
        for (int j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                int tmp = arr[j]; arr[j] = arr[j + 1]; arr[j + 1] = tmp;
                swapped = true;
            }
        }
        if (!swapped) break; // 已有序，提前退出（最好 O(n)）
    }
}
```

### 选择排序

每轮在未排序区间找到最小值，与当前位置交换。交换次数最少（最多 n-1 次），但比较次数固定为 n(n-1)/2，无法提前终止。

```java
public static void selectionSort(int[] arr) {
        int n = arr.length;
        for (int i = 0; i < n - 1; i++) {
            int min = i;
            for (int j = i + 1; j < n; j++) {
                if (arr[j] < arr[min]) {
                    min = j;
                }
            }
            if (min != i) {
                int temp = arr[i];
                arr[i] = arr[min];
                arr[min] = temp;
            }
        }
    }
```

###  插入排序

将每个元素插入左侧已排序区间的正确位置，类似整理扑克牌。对近乎有序的数组效率极高

```java
 public static void insertSort(int[] arr) {
        int n = arr.length;
        for (int i = 1; i < n; i++) {
            int target = arr[i];
            int j = i;
            while (j > 0 && arr[j - 1] > target) {
                arr[j] = arr[j - 1];
                j--;
            }
            arr[j] = target;
        }
    }
```

### 希尔排序

插入排序的改进版，先按较大间隔 `gap` 分组做插入排序，逐步缩小间隔至 1。使元素快速移动到接近最终位置，避免大量逐位移动。

```java
public static void shellSort(int[] arr) {
    int n = arr.length;
    for (int gap = n / 2; gap > 0; gap /= 2) {
        for (int i = gap; i < n; i++) {
            int key = arr[i], j = i;
            while (j >= gap && arr[j - gap] > key) {
                arr[j] = arr[j - gap];
                j -= gap;
            }
            arr[j] = key;
        }
    }
}
```

### 归并排序

分治思想：递归地将数组一分为二，分别排序后合并两个有序子数组。时间复杂度稳定

```java
public static void mergeSort(int[] arr, int l, int r) {
    if (l >= r) return;
    int mid = l + (r - l) / 2;
    mergeSort(arr, l, mid);
    mergeSort(arr, mid + 1, r);
    merge(arr, l, mid, r);
}

private static void merge(int[] arr, int l, int mid, int r) {
    int[] tmp = Arrays.copyOfRange(arr, l, r + 1);
    int i = 0, j = mid - l + 1, k = l;
    while (i <= mid - l && j <= r - l)
        arr[k++] = tmp[i] <= tmp[j] ? tmp[i++] : tmp[j++];
    while (i <= mid - l) arr[k++] = tmp[i++];
    while (j <= r - l)   arr[k++] = tmp[j++];
}
// 调用：mergeSort(arr, 0, arr.length - 1);
```

### 快速排序

选取基准元素 `pivot`，将数组分为小于和大于两部分，递归排序。平均常数因子最小

```java
public static void quickSort(int[] arr, int l, int r) {
    if (l >= r) return;
    int pivot = partition(arr, l, r);
    quickSort(arr, l, pivot - 1);
    quickSort(arr, pivot + 1, r);
}

private static int partition(int[] arr, int l, int r) {
    int pivot = arr[r], i = l - 1;
    for (int j = l; j < r; j++) {
        if (arr[j] <= pivot) {
            i++;
            int tmp = arr[i]; arr[i] = arr[j]; arr[j] = tmp;
        }
    }
    int tmp = arr[i + 1]; arr[i + 1] = arr[r]; arr[r] = tmp;
    return i + 1;
}
// 调用：quickSort(arr, 0, arr.length - 1);
```

### 堆排序

```java
public static void heapSort(int[] arr) {
    int n = arr.length;
    // 建最大堆
    for (int i = n / 2 - 1; i >= 0; i--) heapify(arr, n, i);
    // 逐个提取堆顶
    for (int i = n - 1; i > 0; i--) {
        int tmp = arr[0]; arr[0] = arr[i]; arr[i] = tmp;
        heapify(arr, i, 0);
    }
}

private static void heapify(int[] arr, int n, int i) {
    int largest = i, l = 2 * i + 1, r = 2 * i + 2;
    if (l < n && arr[l] > arr[largest]) largest = l;
    if (r < n && arr[r] > arr[largest]) largest = r;
    if (largest != i) {
        int tmp = arr[i]; arr[i] = arr[largest]; arr[largest] = tmp;
        heapify(arr, n, largest);
    }
}
```
